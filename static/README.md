# Static Files

## Favicon

To add a professional favicon to the site:

1. Create a favicon using a tool like:
   - [Favicon Generator](https://realfavicongenerator.net/)
   - [Favicon.io](https://favicon.io/)

2. Place the following files in this directory:
   - `favicon.ico` (main favicon)
   - `favicon-16x16.png`
   - `favicon-32x32.png`
   - `apple-touch-icon.png`

3. Add this to the `<head>` section via Hugo's head partial if needed.
